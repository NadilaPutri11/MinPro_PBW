# Web Portofolio Dinamis (MinPro 2)

Project ini merupakan pengembangan dari web portofolio statis menjadi dinamis menggunakan PHP dan database.

## Fitur
- Menampilkan data profile dari database
- Menampilkan pengalaman (experiences)
- Menampilkan skill
- Menampilkan sertifikat

## Teknologi
- PHP
- MySQL
- HTML, CSS

## Struktur Folder
- index.php → halaman utama
- conn.php → koneksi database
- assets/ → gambar dan file pendukung
- style.css → styling

## Perbedaan dengan versi sebelumnya
Versi ini sudah menggunakan database, berbeda dengan versi sebelumnya yang masih statis (HTML).
